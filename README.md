🎓 Campus Vault
📌 Overview

Campus Vault is a comprehensive student activity management system developed as an Android application. It provides a centralized platform to manage academic records, attendance, documents, and student activities in a secure and organized manner.
The system replaces traditional and inconvenient ERP systems with a modern, user-friendly, and efficient solution for students, faculty, and administrators.


🚨 Problem Statement

Existing college systems (ERP, manual records, scattered platforms) suffer from:
Poor user experience and complex interfaces.
Fragmented data across multiple systems.
Difficulty in accessing attendance, marks, and documents.
Lack of proper role-based access and security.

💡 Solution

Campus Vault provides:
A single unified platform for all student-related data.
Easy access to attendance, marks, documents, and activities.
Secure system using role-based authentication.
Improved usability compared to traditional ERP systems.

🚀 Key Features:
🔐 Secure login using Firebase Authentication.
👥 Role-Based Access (Student / Faculty / Admin).
📊 Attendance tracking system.
📝 Marks and academic performance management.
📁 Document upload, storage, and access.
📢 Activity and event tracking.
☁️ Cloud-based data management.
⚡ Fast and user-friendly interface.
